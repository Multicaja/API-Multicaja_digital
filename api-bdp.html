Bienvenido a la API de Multicaja (Alpha) Este es un proyecto alpha y por lo tanto sufrirá muchos cambios Todos los requests son autenticados usando un api-key. Existen dos ambientes: qa y producción. El host de qa es apiqa.multicaja.cl.
